## Introduction
 A one stop solution for all your notes - Notepedia is a note-taking application that is rich with features.
 
 [Check it out !](https://notepedia.netlify.app/)

## Features 

1. Bolden Text
2. Italicize Text
3. Underline Text
4. Copy Note 
5. Responsive Design
